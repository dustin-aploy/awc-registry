# AWC Registry Docs

This folder contains the main guides for submitting and reviewing AWC worker listings.

`awc-registry` is source-available under BUSL-1.1 because this repository sits closer to registry operation and commercial control concerns than the spec and devkit repositories.

Start here:

- `licensing.md` for license scope and rationale
- `submit-a-worker.md` for the submission checklist
- `after-submission.md` for the verification and listing flow
- `listing-lifecycle.md` for listing states
- `certification-process.md` for stronger evidence and review requirements
